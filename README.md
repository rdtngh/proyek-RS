# HETS Prototype

Prototype website pelatihan karyawan rumah sakit berbasis HTML, CSS, dan JavaScript vanilla.

## Struktur Utama
- index.html — landing page
- login.html — halaman login demo
- dashboard.html — ringkasan progress pelatihan
- materi.html — daftar modul dan progress belajar
- quiz.html — kuis interaktif 10 soal

## Teknologi
- HTML5
- CSS3
- Vanilla JavaScript
- Data statis dari js/data.js

## Cara Menjalankan
### Opsi 1: Satu klik di Windows
Klik file berikut:
- run.bat

File ini akan otomatis:
- menjalankan server lokal
- membuka halaman website di browser

### Opsi 2: Manual lewat terminal
Jalankan perintah berikut:

```bash
python -m http.server 8000
```

Lalu buka http://localhost:8000/
